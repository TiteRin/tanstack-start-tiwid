# Tada Tiwid

## English

### Purpose
Tada Tiwid is a warm, cozy, and non-judgmental space where people can log the tasks and wins of their day. This is not a Todo List. It is a Done List, a Tada List, built to celebrate progress and help people feel better.

I am building this because classic Todo Lists can add pressure, especially for people who are depressed, neuroatypical, anxious, etc. Yet checking boxes can still help us move forward, feel better, and recognize that we do things.

I also wanted to keep my hands on a React environment, discover the TanStack Start framework, continue a TDD approach, and experiment with hexagonal architecture.

### Tech
- React 19
- TanStack Start + TanStack Router
- Vite
- Tailwind CSS
- Prisma (SQLite)
- Better Auth
- Vitest + Testing Library
- Storybook
- Biome (lint/format)

### Run the project
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```

### Tests
```bash
npm run test
```

### Lint and format
```bash
npm run lint
npm run format
npm run check
```

### Storybook
```bash
npm run storybook
```

## Français

### Objectif
Tada Tiwid est un espace chaleureux, cozy et sans jugement où l’on peut saisir les tâches et les succès de sa journée. Ce n’est pas une Todo List. C’est une Done List, une Tada List, faite pour célébrer et se sentir mieux.

Je développe ce projet car une Todo List peut apporter beaucoup de pression, surtout chez les personnes dépressives, neuroatypiques, anxieuses, etc. Pourtant, cocher des cases aide parfois à avancer, à aller mieux, et à se rendre compte qu’on fait des choses.

Je voulais aussi garder la main sur un environnement React, découvrir le framework TanStack Start, continuer l’approche TDD, et essayer l’approche hexagonale.
